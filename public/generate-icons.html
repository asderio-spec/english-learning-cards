PWA 아이콘 생성

아이콘 생성 방법:

  1. 아래 SVG 이미지들을 우클릭하여 저장
  2. 온라인 SVG to PNG 변환기 사용 (예: convertio.co, cloudconvert.com)
  3. 각각 필요한 크기로 변환:
    • pwa-192x192.png (192x192)
    • pwa-512x512.png (512x512)
    • apple-touch-icon.png (180x180)
    • favicon-32x32.png (32x32)
    • favicon-16x16.png (16x16)
  4. 생성된 PNG 파일들을 public 폴더에 저장

자동 생성 (Node.js 환경):

더 정확한 PNG 생성을 위해 다음 패키지 설치 후 사용:

npm install sharp

그 후 이 스크립트를 수정하여 sharp를 사용한 자동 변환 구현
